# 🚀 Flask Blog Web Application

A Flask-based blog web application with user authentication and CRUD operations using MySQL and Bootstrap.

---

## 📌 Overview

This project is a full-stack blog application that allows users to register, login, and manage blog posts.

---

## 🎯 Features

- User Registration & Login  
- Admin Dashboard  
- Add / View / Update / Delete Posts  
- MySQL Database Integration  
- Responsive UI using Bootstrap  

---

## 🛠️ Tech Stack

- Python Flask  
- MySQL  
- HTML, Bootstrap  
- Git & GitHub  

---

## ⚙️ Run Locally

pip install flask mysql-connector-python werkzeug  

CREATE DATABASE flaskblog;  

py app.py  

http://127.0.0.1:5000
